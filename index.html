In this game you will play as either the ATTACKER or DEFENDER.

These will be chosen at random

Players will take turns placing the numbers 1-9 into the grid.

1 2 3 4 5 6 7 8 9

Each number can only be used once.

The aim for the ATTACKER is to get three numbers in a row to add to exactly 15.

This can be done in any direction

The ATTACKER has won the game in the example above"

The aim for the DEFENDER is to stop the attacker from getting a total of 15.

The DEFENDER wins once all the numbers ave been used and none of the rows, columns or diagonals add to exactly 15.

The DEFENDER has won the game in the example above"

There are so many amazing strategies to discover.

If you lived 1000 years and played this game every single day you still wouldn't have seen every possibility.

Good Luck!!
